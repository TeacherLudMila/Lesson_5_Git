Работа в консоли

  • ls - вывести список содержимого папки, в которой находимся
  • pwd - вывести путь, по которому мы сейчас находимся
  • mkdir new_folder - создать папку “new_folder”
  • rmdir new_folder - удалить папку “new_folder”
  • rm –rf new_folder - удалить папку со всем ее содержимым
  • cd ../.. - выйти из папки на 2 уровня вверх
  • cd lessons/new_folder - зайти в папку “new_folder”
  • touch index.html style.css - создать index.html и style.css
  • code index.html style.css - создать и сразу открыть в редакторе кода index.html и style.css

Работа с Git и GitHUB. Основные команды.

  • git init - инициализировать репозиторий
  • git status - показать текущий статус
  • git add . - добавить все изменения
  • git add index.html - добавляет только index.html
  • git commit -m "first commit" - закоммитить все изменения (название коммита - "first commit")
  • git branch -M main - создаем ветку с названием main (основная ветка)
  • git remote add origin https://github.com/... - связываем друг с другом локальный и удаленный репозиторий
  • git push -u origin main - заливаем все на удаленный репозиторий на ветку main
  • git pull - забирает изменения с удаленного репозитория в локальный
  • git clone - клонирует проект с удаленного репозитория

Заливаем изменения

  • git status - проверяем статус (изменения)
  • git add . - добавляем все изменения
  • git status - проверяем статус (что произошло)
  • git commit -m "add... change..." - коммитимся
  • git push -u origin main - заливаем на ветку все изменения

Работа с ветками

  • git branch - показывает список веток
  • git branch branch-name - создает новую ветку с названием "branch-name"
  • git branch -D branch-name - удаляет ветку с названием "branch-name"
  • git checkout - переключиться на другую ветку
  • git checkout branch-name - переключается на последний коммит в ветке "branch-name"
  • git checkout -b branch-name - создает и переключается на ветку "branch-name"
  • git merge branch-name - совмещает текущую ветку с "branch-name"

Конфигурация и параметры git

  • git config --global user.name - показывает имя пользователя
  • git config --global user.name 'new user' - устанавливает глобальное имя пользователя
  • git config --local user.name 'new user' - устанавливает локальное имя пользователя
  • git config --global user.email- показывает email пользователя
  • git config --global user.email 'email@email.com' - устанавливает глобальный email пользователя
  • git config --local user.email 'email@email.com' - устанавливает локальный email пользователя
  • git config --list - выводит список настроек
  • git config --global --unset-all user.name - удалить глобальные настройки имени
  • git config --global --unset-all user.email - удалить глобальные настройки почты
